# Live Streamer

Create a locally accessible live stream of your webcam.

## Usage

1. Set up a venv and install the requirements:
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

2. Run the server:
```bash
python3 main.py
```
